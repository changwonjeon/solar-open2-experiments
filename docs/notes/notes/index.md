# Notes Directory

Ideas, thoughts, log entries, and general notes.

## Categories

* Ideas - New concepts and brainstorming
* Log - Chronological activity logs
* Thoughts - Reflections and observations
* Questions - Open questions and inquiries

## Log Format

Each log entry should be a separate file with OKF format:

```markdown
---
type: LogEntry
title: <Entry Title>
description: <Brief summary>
tags: [log, idea]
timestamp: 2026-07-17T00:00:00Z
---

# Entry

Detailed content of the log entry or note.

# Related

* [Related Note](../related-nte.md)
```

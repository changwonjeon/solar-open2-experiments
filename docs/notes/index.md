# Notes Directory

Wiki-style knowledge base using LLM-Wiki structure.

## Categories

* [People](people/index.md) - Researchers, engineers, and contributors
* [Models](models/index.md) - Model architectures, techniques, and implementations
* [Papers](papers/index.md) - Research papers, summaries, and reviews
* [Projects](projects/index.md) - Active and completed projects
* [Notes](notes/index.md) - Ideas, thoughts, and log entries
* [Writing](writing/index.md) - Essays, blog posts, and articles

## How to Use

1. Browse categories above or use the search function
2. Each note follows OKF format with YAML frontmatter
3. Link between notes using markdown links: `[Note Title](../category/note-title.md)`
4. Add new notes by creating `.md` files in the appropriate category directory

# Submission Contract

Submit one anonymous Technical Report PDF, Title, and Abstract before the event hard cut. Use the official ICML 2026 style and keep report body content within four pages; references and appendix are excluded from that body limit.

The event review form requires Soundness 1–4, Presentation 1–4, Significance 1–4, Originality 1–4, Overall Recommendation 1–6, Confidence 1–5, and a non-empty constructive Comment. The frozen local result also preserves upstream Contribution and evidence trace.

Freeze the review-agent name/version, prompt and schema hashes, paper/evidence hashes, output hashes, timings, failures, and receipts. Ensure every report number is recomputable from saved evidence. Label mock runs as mock; never present them as live platform performance.

Hard-fail submission artifacts containing a participant name, account handle, local absolute path, email, personal repository URL, author or acknowledgment text, or placeholders. Confirm Title and Abstract match the final report claims.

# Meeting Minutes Experiment Directory

Solar Open2가 Claude Code CLI로서 회의록 작성에 얼마나 뛰어난지 검증하기 위한 실험 결과물을 보관합니다.

## 실험 개요

- **목표**: Solar Open2 모델이 회의록(회의 음성/텍스트 → 구조화된 OKF 포맷 문서)을 얼마나 정확하고 구조적으로 작성하는지 검증
- **1차 실험 (랄프톤/랄프루프)**: Codex CLI의 자율 루프 실행을 Solar Open2 + Claude Code로 재현 → 현재 완료, 후속 작업 대기
- **2차 실험 (회의록 작성)**: Solar Open2의 자연어 이해, 정보 추출, 구조화 작성 능력을 회의록 작성 태스크로 데모
